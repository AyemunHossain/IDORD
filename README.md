## Installation

IDORD requires Python3 and pip to run.

Install the dependencies and start the tool.

```sh
pip install -r requirements.txt

#Active the virtual env [varies in linux and windows]

RUN: cd Wrapper 
RUN: python3 IDORD.py
#bang bang
```
